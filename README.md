# Expense Tracker

CLI tool to track daily expenses with category-wise analysis.

## Features
- Add expenses with amount, category, description
- View all expenses
- Calculate total spending
- View spending by category
- Persist data to CSV

## Usage
```bash
python expense_tracker.py
```

## Tech Stack
- Python 3.9+
- CSV module for data storage
- datetime for timestamps

## Sample Output
```
Total Spending: ₹5,420
Food: ₹2,100
Transport: ₹800
Entertainment: ₹1,520
```

## What I Learned
- Working with CSV files
- Data aggregation
- Dictionary operations for grouping
